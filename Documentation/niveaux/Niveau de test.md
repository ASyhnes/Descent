### scénario: 


1.  **Un slot mémoire :** 
	1. **scénario :** un personnage te dis: "il serait bien que tu ailles voir le superviseur". le joueur doit aller voir le superviseur qui lui fera passer au niveau de slot 2. mais d'autre personnage viennent perturber le joueur qui ne se rappelle plus ce qu'il doit faire.
	   
	2. **porte :** le superviseur "je suis le superviseur de ce niveau. Que me voulez vous ?"
	   
	3. **condition :**  portrait du superviseur dans le slot 1. Il s'obtient en parlant à l'ouvrier1_maritime. si condition activé : "Ha c'est vous le nouveau commis? parfait. Je compte sur votre mémoire. " puis "félicitation, vous avez débloquer un deuxième slot mémoire." puis **récompense**. "je vous invite à rejoindre votre Hub. C'est la troisième porte du troisième étage. Vous trouverez la nomenclature d'ouverture de porte sur l'affiche à l'entré de l'étage "
	   
	4. **activation :** parler a ouvrier1_maritime qui dis "Allez voir le superviseur " pour avoir le slot mémoire avec le portrait du superviseur.
	   
	5. **note :** le personnage se déplace aléatoirement. il n'est visible que dans la zone de lumière. D'autre personnages donnent d'autre objectifs "buy" par exemple. 
	   
	6. **récompense :** Deuxième slot mémoire. 
	
2. **Deux slots mémoire :** 
	1.  **scénario :** une affiche dit "pour obtenir votre code d'activation de Hub, référez vous à la nomenclature suivante : Les 3 premières lampes affiche votre code d'étage, et les 3 secondes affiche votre numéro de porte. Mémorisez le code de lampes correspondante à votre étage puis la lampe correspondante à votre porte"
	   
	2. **porte :** la porte du hub 
	
	3. **condition :**  deux slot mémoire remplit avec les deux codes des lampe correspondant étage/porte
	   
	4. **activation :** regarder la lampe du troisième étage puis la lampe de la troisième porte
	   
	5. **note :** le message sur la porte du Hub dit "Entrez votre code ... "  puis "Code Invalide/ Code exécuté correctement."
	   
	6. **récompense :** Accès au hub.

3. **Deux slots mémoires + énigme sans indice:**  
4. Autres niveaux (3 slot>5 slot)/augmentation du niveau de conscience/changement de vétement
